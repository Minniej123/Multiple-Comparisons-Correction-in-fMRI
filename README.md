# Multiple-Comparisons-Correction-in-fMRI
An evaluation of multiple comparisons correction in fMRI (including an evaluation of satellite imaging methods and their appropriateness)

This Github includes:


- Simulation

  - Simulation plots 
- Real data
  - Real data code

The Haxby data set (used for Real data analysis) was downloaded in Python using: 
nilearn.datasets.fetch_haxby(data_dir=None, subjects=(2,), fetch_stimuli=False, url=None, resume=True, verbose=1)
